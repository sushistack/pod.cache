## 2025-12-31 Transformer의 탄생과 지능의 미스터리

<details>
<summary>Transformer, 프롬프트 엔지니어링, Context Engineering, AI Agent</summary>

### Transformer의 등장

Transformer 이전에는 순환 신경망(Recurrent Neural Network, RNN)의 한 종류인 LSTM(Long Short-Term Memory)이 시퀀스 데이터 처리에 널리 사용되었음.

### 프롬프트 엔지니어링(Prompt Engineering) and Context Engineering

- 프롬프트 엔지니어링은 LLM에 더 나은 입력 시퀀스를 제공하여, 출력 시퀀스가 우리가 해결하려는 목표에 더욱 정확하게 부합하도록 만드는 것
- Context Engineering은 LLM에 더 나은 입력 시퀀스를 제공하여, 출력 시퀀스가 우리가 해결하려는 목표에 더욱 정확하게 부합하도록 만드는 것
    - 비즈니스/산업 특정 정보: 기업 내부 데이터, 고객 정보, 특정 산업 용어 등을 프롬프트에 효과적으로 통합합니다.
    - 외부 도구(Tools) 활용: LLM이 특정 작업을 수행하기 위해 외부 API나 데이터베이스, 검색 엔진 등과 상호작용하도록 하는 메커니즘을 제공합니다. (예: 최신 정보 검색, 계산 수행, 이미지 생성 등)

### 자율성을 갖춘 AI Agent

워크플로우 제어형: LLM이 전체 워크플로우를 책임지고, 다음 발생할 일을 결정하며, 외부 도구나 기능을 호출하는 시스템.
반복 루프형: LLM이 반복적으로 호출되는 루프 내에서 작동하며, 외부 도구에 접근하여 다양한 행동을 취할 수 있는 시스템. (예: Claude Code, GPT Agent)
