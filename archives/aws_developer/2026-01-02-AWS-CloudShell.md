## 2026-01-02 AWS CloudShell

<details>
<summary>복습 키워드: AWS CloudShell의 핵심 기능과 활용</summary>

### AWS CloudShell이란?

AWS Management Console에서 직접 접근할 수 있는 브라우저 기반의 셸 환경.

### 주요 기능 및 장점

- 사전 설치된 AWS CLI 및 자동 인증
- 영구 스토리지 제공
- 편리한 파일 업로드 및 다운로드 기능
- UI 커스터마이징 및 다중 터미널 지원

### CloudShell 사용 시 고려사항

- 리전 가용성
- 기존 터미널과의 병행 사용
